# arduino-tetris
Implementation of game similar to tetris for arduino with tft display and 5-way tactile switch.
# Used components
- Arduino pro mini
- 1.8 128x160 TFT LCD with SPI Interface
- 5-way tactile switch
- one Resistor 2.2k Ohms 5%
